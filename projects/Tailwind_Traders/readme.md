# Power BI Analytics Project: Tailwind Traders

## Overview
Welcome to the **Tailwind Traders Power BI Analytics Project**!  
This repository serves as a comprehensive resource hub for immersing in the world of data analysis tailored for Tailwind Traders, a fictitious company.  

As a Power BI analyst, I embarked on a data-driven journey to address key business questions and uncover valuable insights within the Tailwind Traders ecosystem.

---

## Project Objectives
- Gain hands-on experience in leveraging **Power BI analytics** for meaningful business insights.  
- Navigate real-world tasks, engaging with diverse scenarios and stakeholders.  
- Deliver a polished and insightful **Power BI report** showcasing proficiency in data visualization and analysis.

## Project Phases

### 1. Data Preparation & Configuration
- Downloaded and cleaned sales data.  
- Calculated **Gross Revenue, Total Tax, and Net Revenue**.  
- Loaded and transformed data using **Power BI and Power Query**.  
- Integrated **historical currency exchange data**.  

### 2. Data Model Development
- Established relationships between tables (*Countries, Sales, Purchases*).  
- Configured a **Calendar table** for time intelligence.  
- Created calculated tables for **sales in USD**.  

### 3. Aggregations & Report Generation
- Configured DAX measures for:  
  - Yearly Profit Margin  
  - Quarterly Profit  
  - Year-to-Date (YTD) Profit  
  - Median Sales  
- Developed **Sales Overview** and **Profit Overview** reports with relevant visualizations.  

### 4. Executive Dashboard, Alerts & Subscriptions
- Designed an **Executive Dashboard** with core KPIs and drill-downs.  
- Configured **mobile view** for accessibility.  
- Implemented a **daily alert** for Gross Revenue (USD < $400).  
- Created **subscriptions** for Sales and Profit Overview reports.
